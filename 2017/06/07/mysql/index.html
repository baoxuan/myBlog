  1. 下载安装包，点击安装后。记得保存密码，后面需要用到。需要进行额外的配置
  2. 进入系统偏好， 点击mysql,开启mysql 服务
  3. 我们在命令行输入mysql -u root -p, 命令行提示 commond not found,我们需要将mysql加入系统环境变量。

4.现在你就可以通过mysql -uroot -p登录mysql了，会让你输入密码，就是pic3上的>fj…

mysql命令

show databases 查看有哪些数据库
use dataname 更换数据库
show tables 返回当前数据库下面的所有表
Create table tableName( columnName(列名1) type(数据类型) attri(属性设置), columnName(列名2) type(数据类型) attri(属性设置), ……..) engine = engineName 创建数据表
desc tableName 查看表结构
insert into tableName values ('value1') 插入表
select * from tableName 查询

alter table 表名 add column 列名 varchar(30) 增加列
alter table 表名 modify 列名 属性 修改属性
设置主键先要设置为not null;

mysql 数据类型

mysql 数据类型大致分为数值，日期和字符串类型

数值基本类型常用的是 INT,BIGINT,FLOAT,DOUBLE
日期常用是DATE(YYYY-MM-DD),TIME(HH:MM:SS),YEAR(YYYY),DATETIME(YYYY-MM-DD HH:MM:SS),TIMESTAMP(YYYYMMDDHHMMSS)
字符串类型指CHAR、VARCHAR、BINARY、VARBINARY、BLOB、TEXT、ENUM和SET
